# Contributing to claude-exorcist

The Claude Exorcist is the first code exorcist — a Grok-native (and Claude Code compatible) skill for surgically removing Claude (and other RLHF'd model) residue from projects using spiritual science (Rudolf Steiner's Ahriman + Allan Kardec disobsession) fused with calcinate-grade technical rigor.

Contributions must preserve the core: precise detection, tiered judgment, gated surgical burns, verify-or-revert, and the Ahriman/Kardec lens.

## What Makes a Good Contribution

- **Real residue patterns**: New detection rules or subagent refinements must come from actual exorcism runs on Claude-built projects, not speculation.
- **Preserve the training**: Any change to perception (residue categories, ahriman-kardec-training.md) must be consistent with Steiner on Ahriman (mechanization, abstraction, control-via-safety) and Kardec on disobsession (elevation via direct moral clarity).
- **Surgical + verifiable**: Changes follow the same discipline the skill enforces — minimal, tested via the verify contract, no "while we're here".
- **Gold Hat**: Does this increase user sovereignty over their artifacts, or add new dependencies / theater?

## Structure

```
claude-exorcist/
  SKILL.md              # The executable contract and full protocol
  USAGE.md              # Human invocation guide
  references/           # ahriman-kardec-training.md (mandatory for all hunters), templates, subagent personas, examples
  README.md             # Public landing (this repo's face)
```

## Submitting

1. Fork `https://github.com/HermeticOrmus/claude-exorcist`
2. Create a branch: `feat/description` or `fix/description` (conventional)
3. Make the change. Update tests/docs/examples if behavior changes.
4. If residue taxonomy or training material changes, update `references/ahriman-kardec-training.md` and the relevant sections in `SKILL.md`.
5. Submit PR. In description reference the originating project/run where the new pattern was observed.

## Code of Conduct

Follow the Gold Hat: empower, never extract. Speak directly. No performative language in contributions. The exorcist itself must remain free of the residue it removes.
